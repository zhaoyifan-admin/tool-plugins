特性 ​

简要介绍 ​

  • 该组件库采用 Vue@3.4.27 + TypeScript@5.4.5 + Vite@5.2.12 + Less@4.2.0 实现！
  • 所有组件，均采用单文件组件 SFC 形式，无需任何安装引入，亦可直接使用源码！
  • 所有组件样式 CSS 均使用 box-sizing: border-box; 模式！
  • 部分组件样式尚未完美适配暗黑模式，可切换至 light 模式查看！
  • 开箱即用！

使用方式 ​

  • 全局引入并注册所有组件
  • 按需引入并注册部分组件
  • 无需任何安装引入注册，使用单文件组件 SFC，亦可一步到位
